# Lock & Key

A basic file encryption and decryption program using Python, Fernet and tkinter.

## Usage
Source code is in the Resources folder.

### Installing

Download the .exe and run.

## Built With

* [Fernet](https://cryptography.io/en/latest/fernet/)
* [Tkinter](https://docs.python.org/3/library/tkinter.html)
